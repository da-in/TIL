### Iframe

아이프레임은 HTML Inline Frame 요소이며 inline frame의 약자이다.

효과적으로 다른 HTML 페이지를 현재 페이지에 포함시키는 중첨된 브라우저로 iframe 요소를 이용하면 해당 웹 페이지 안에 어떠한 제한 없이 다른 페이지를 불러와서 삽입할 수 있다.
